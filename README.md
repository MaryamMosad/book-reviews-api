# Book Reviews System

## Description

An api where:

Admins can add authors and their books which can belong to multiple different genres.

Users can view books information and add reviews.

## Stack Used:

- NodeJs
- Express.js
- TypeScript
- GraphQL
- PostgreSQL
- sequelize ORM
